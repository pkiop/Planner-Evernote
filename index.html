에버노트 API 만지기


boost lib와 아파치의 thrift를 사용해야 한다. 

boost lib : https://www.boost.org/users/history/version_1_69_0.html
아파치 thrift : https://github.com/apache/thrift


  • Evernote API키 받고 샘플 코드 다운받아 실행시켜보려함
  • visual studio 프로젝트는 win32로 생성, precompile된 헤더 사용안함 하라고 해서 했음 (빈프로젝트와 다른게 뭔지)


부스트라이브러리 쓰기위해

  • 부스트 라이브러리는 압축 다운받고 VS2017 developer command prompt로 boostrap.sh 실행시키면 b2.exe파일 생성됨
  • b2 toolset=msvc-14.1 variant=debug,release link=static threading=multi address-model=32 runtime-link=shared 입력하면됨. 14.1은 VS2017
  • 위 처럼 나오면 성공 
  • 샘플 코드 빌드해서 에러나는 라이브러리들 다 추가해주기
  • 링커 -> General -> additional library directories 에서 stage\lib 추가해주기
